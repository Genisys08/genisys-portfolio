# Genisys Graphics — V2.0 (The Snyder Cut)

Operatic, pitch-black portfolio for a high-end graphics design studio.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## V2.0 features
- True WebGL stylized "liquid metal" shader (Three.js) over pitch-black void
- Ambient golden-ember particle engine
- Dead-center preloader (Grid)
- Glowing gold music vibe toggle (top-right, z-index 90)
- Text-scramble decryption on headers and tabs
- Cinematic letterbox transitions on category filter
- Magnetic UI elements (cursor + touch)
- Directional gold drop-shadows driven by gyroscope/cursor
- Isolate Focus Mode (background dims to 10% on card hover)
- Bulletproof error boundary + WebGL detection with CSS fallback
- Fixed liquid-glass WhatsApp CTA bar
- Lenis smooth scroll, IntersectionObserver lazy images
- Full OG / Twitter meta tags
